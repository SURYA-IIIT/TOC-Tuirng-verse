#  TuringVerse — Turing Machine Simulator

##  Project Overview
**TuringVerse** is an interactive web-based application that simulates the working of **Turing Machines** with rich visualizations. It helps in understanding core concepts of **Theory of Computation** through animation, step-by-step execution, and graphical representation.

This project demonstrates how abstract computational models can be visualized and explored interactively.

---

##  Objectives
- Simulate the working of a Turing Machine step-by-step  
- Visualize tape movement and state transitions  
- Understand different variants of Turing Machines  
- Demonstrate the concept of Universal Turing Machine  
- Explore language acceptance (Recursive & Recursively Enumerable)  

---

##  Concepts Covered

###  Basic Turing Machine
- Formal definition:  
  **M = (Q, Σ, Γ, δ, q₀, q_accept, q_reject)**  
- Transition-based computation  
- Step-by-step execution  

---

###  Tape Simulation
- Infinite tape representation  
- Dynamic expansion  
- Head movement (Left / Right)  
- Real-time updates  

---

###  State Diagram Visualization

- Highlights:
  - Current state

###  Variants of Turing Machine
- Multi-Tape Turing Machine  
- Non-Deterministic Turing Machine  
- Multi-Track Turing Machine  
- Two-Way Infinite Tape  
- TM as Integer Computer  

---

###  Universal Turing Machine (UTM)
- Simulates any Turing Machine  
- Input:
- Machine description  
- Input string  
- Demonstrates:
> One machine can simulate all machines  

---

###  Language Acceptance
- Recursively Enumerable Languages  
- Recursive (Decidable) Languages  

✔ Accept → String belongs to language  
✔ Reject → String not in language  
✔ Loop → Represents RE behavior  

---

###  Church–Turing Thesis
> Any computation that can be performed by an algorithm can be computed by a Turing Machine.
